# Portfolio

Personal portfolio built with **React**, **TypeScript**, and **Vite**.

## Stack

- **React 19** + **TypeScript**
- **Vite** for build tooling
- **React Router** for routing
- Deployed via **Vercel**

## Getting Started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

## Project Structure

```
src/
├── components/
│   ├── sections/   # Hero, About, Skills, Contact
│   └── ui/         # Reusable UI (Section, TiltCard)
├── data/           # Content (projects, skills, stats)
├── hooks/          # Custom hooks (useScrollReveal, useTilt)
├── pages/          # Route pages (Home, ProjectsPage)
└── types/          # TypeScript interfaces
```
